# 🎬 AI-Cut Pro 使い方ガイド

このツールは、動画を1本用意するだけで、以下の処理をブラウザ上のGUI画面から直感的に行える動画編集支援アプリケーションです。

1. **無音部分の自動カット**（ジェットカット）
2. **高精度文字起こし**（Whisper）
3. **テロップのリアルタイムプレビュー**（Remotion）
4. **ブラウザ上での直接テロップ修正**（Magic Editor）
5. **Premiere Pro 用データの書き出し**（XML, SRT）

---

## 🚀 起動方法

### 1. サーバーの起動
ダッシュボード（Streamlit）、プレビュー（Remotion）、保存API（Flask）の3つを起動します。

**ターミナル1: APIサーバーの起動（ポート5001）**
```bash
source venv/bin/activate
python api_server.py
```

**ターミナル2: プレビュー画面の起動（ポート3001）**
```bash
cd remotion-project
npm run dev -- --port 3001
```

**ターミナル3: ダッシュボードの起動（ポート8501）**
```bash
source venv/bin/activate
streamlit run app.py
```

---

## 💻 使い方・ワークフロー

### Step 1: 動画の処理を開始する
1. ブラウザで `http://localhost:8501`（ダッシュボード）を開きます。
2. **「🚀 編集実行」タブ** を選択します。
3. 左側のサイドバーでテロップの文字数やフォントサイズなどの基本設定を行います。
4. メイン画面のアップロード枠に動画ファイルをドラッグ＆ドロップし、「🎬 無音カット＆テロップ生成を開始」ボタンを押します。
5. （動画の長さに応じて数分かかります。）

### Step 2: テロップの一括修正
1. 処理が完了したら、**「🖋️ テロップの直接編集」タブ** を開きます。
2. Whisperが生成した文字起こしデータがスプレッドシート形式で表示されます。
3. 明らかにおかしい漢字変換などをここでダブルクリックして一気に修正し、「💾 変更を保存して反映する」を押します。

### Step 3: プレビューを見ながら微調整（魔法のエディタ）
1. **「📺 リアルタイムプレビュー」タブ** を開きます。
2. 再生ボタンを押して動画を確認します。
3. 気になるテロップがあれば、**画面上の文字を直接クリック**して自由にタイピングで修正できます。
4. 修正後、Enterキーを押すか枠外をクリックすると、緑色の枠が光って自動的にファイルに保存されます。

### Step 4: 最終出力
編集が完了したら、「🚀 編集実行」タブに戻り、編集済みの動画（MP4）や、Premiere Pro用のXMLファイル、SRT字幕ファイルをダウンロードしてご自身の編集フローに役立ててください。

---

## 📁 成果物について

- **`output/final.mp4`**: 自動カットとテロップ焼き付けが完了した動画。
- **`output/premiere_sequence.xml`**: Premiere Pro上でカット編集を微調整するためのシーケンスデータ。
- **`output/subtitles.srt`**: Premiere Proに「キャプショントラック」としてそのまま読み込める高品質な字幕ファイル。
